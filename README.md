# 🧠 Portafolio de Aprendizaje: Análisis de Datos (Cisco Academy)

**Descripción**
Este repositorio contiene proyectos académicos, laboratorios prácticos y scripts de análisis desarrollados durante mi formación en la Academia de Cisco. Refleja el dominio de conceptos estadísticos, consultas de bases de datos y manipulación de información.

**✨ Contenido del Repositorio**
* **Consultas SQL:** Scripts para la extracción, filtrado y limpieza de datos estructurados.
* **Modelado de Datos:** Ejercicios prácticos de estructuración de bases de datos para optimización de reportes.
* **Casos de Estudio:** Resolución de problemas de negocio aplicando técnicas de análisis descriptivo.

**🛠️ Tecnologías e Institución**
* **Academia:** Cisco Networking Academy
* **Tecnologías:** SQL, Excel, Fundamentos de Análisis de Datos

**🚀 Objetivo del Proyecto**
Consolidar las habilidades técnicas requeridas para el rol de Analista de Datos, aplicando la teoría educativa a escenarios prácticos del mundo empresarial.
